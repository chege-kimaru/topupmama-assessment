# TopUpMama Assessment

[Visit App](topupmamaassessment.netlify.app)

## Tech Used

- Angular 11
- NgRx
- Bootstrap 4
- Ngx-toatr
- SweetAlert 2

## Installation

1. `git clone https://github.com/chege-kimaru/topupmama-assessment.git && cd topupmama-assessment`
2. `npm install`
3. `ng s -o`
